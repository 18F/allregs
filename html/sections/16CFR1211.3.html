CFR /  Title 16  /  Part 1211  /  Sec. 1211.3 Units of measurement.

If a value for measurement is followed by a value in other units, in parentheses, the second value may be only approximate. The first stated value is the requirement. [57 FR 60455, Dec. 21, 1992, as amended at 65 FR 70657, Nov. 27, 2000]
