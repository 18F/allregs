CFR /  Title 48  /  Part 919  /  919.7013 Reports.

(a) Prior to performing an evaluation of a Mentor's performance under its Mentor-Protege Agreement for use in award fee evaluations, the Mentor-Protege Program Manager must consult with the cognizant DOE technical program manager and must provide a copy of the performance evaluation comments regarding the technical effort and Mentor-Protege development to the contracting officer.

(b) The DOE Mentor-Protege Program Manager must submit semi-annual reports to the cognizant contracting officer regarding the participating Mentor's performance in the Program for use in the award fee determination process.

(c) The Mentor firm must submit progress reports to the DOE Mentor-Protege Program Manager semi-annually.
