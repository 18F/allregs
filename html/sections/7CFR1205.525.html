CFR /  Title 7  /  Part 1205  /  Sec. 1205.525 Entry of gin code number.

The warehouse that first receives a bale for storage after ginning shall enter the gin code number of the gin at which the bale was ginned on the warehouse receipt issued for the bale. [57 FR 29192, July 1, 1992]
