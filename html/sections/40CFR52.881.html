CFR /  Title 40  /  Part 52  /  Sec. 52.881 PM [bdi1][bdi0] State implementation plan development

The state has submitted a committal SIP for Kansas City, Kansas. The committal SIP contains all the requirements identified in the July 1, 1987, promulgation of the SIP requirements for PM10 at 52 FR 24681, except the state will report the PM10 data which exceed the standard within 60 days of the exceedance, rather than 45 days. [55 FR 1423, Jan. 16, 1990]
