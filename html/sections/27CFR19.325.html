CFR /  Title 27  /  Part 19  /  Sec. 19.325 Change of packages.

A proprietor may transfer spirits or wines in storage from one package to another. Each new package must contain spirits from only one package except in the case of spirits of 190 deg. or more proof. Packages of spirits must be marked as provided in subpart S of this part. Each package of wine must bear the same marks as the package from which the wine was transferred. (26 U.S.C. 5201)
