CFR /  Title 18  /  Part 286  /  Sec. 286.106 Form and style.

Each copy of such memorandum must be complete in itself. All pertinent data should be set forth fully, and each memorandum should set out the facts and argument as prescribed for briefs in Sec. 385.706 of this chapter.
