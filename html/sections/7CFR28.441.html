CFR /  Title 7  /  Part 28  /  Sec. 28.441 Strict Middling Yellow Stained Color.

Strict Middling Yellow Stained Color is color which is deeper than that of Strict Middling Tinged Color. [57 FR 34498, Aug. 5, 1992]
