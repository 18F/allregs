CFR /  Title 20  /  Part 619  /  Sec. 619.4 Data exchange standardization for the UI Benefits and

(a) XML is the data exchange standard for the real time ICON applications set out in Sec. 619.2 and for the SIDES exchanges set out in Sec. 619.3 associated with major IT modernization projects, to upgrade UI Benefits and Tax Systems by SWAs using Federal funds.

(b) The standard designated in paragraph (a) of this section is effective March 21, 2014.
