CFR /  Title 47  /  Part 61  /  Sec. 61.83 Consecutive numbering.

Issuing carriers should file tariff publications under consecutive FCC numbers. If this cannot be done, a memorandum containing an explanation of the missing number or numbers must be submitted. Supplements to a tariff must be numbered consecutively in a separate series. [76 FR 43216, July 20, 2011]
