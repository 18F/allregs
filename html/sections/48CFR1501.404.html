CFR /  Title 48  /  Part 1501  /  1501.404 Class deviations.

Requests for class deviations to the FAR and the EPAAR shall be submitted to the HCA for processing in accordance with FAR 1.404 and this section. Requests shall include the same type of information prescribed in 1501.403 for individual deviations. [67 FR 5072, Feb. 4, 2002]
