CFR /  Title 40  /  Part 96  /  Sec. 96.40 State trading program budget.

The State trading program budget allocated by the permitting authority under Sec. 96.42 for a control period will equal the total number of tons of NOX emissions apportioned to the NOX Budget units under Sec. 96.4 in the State for the control period, as determined by the applicable, approved State implementation plan.
