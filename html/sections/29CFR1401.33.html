CFR /  Title 29  /  Part 1401  /  Sec. 1401.33 Description of information requested.

(a) Each request should reasonably describe the records being sought, in a way that they can be identified and located. A request should include all pertinent details that will help identify the records sought.

(b) If the description is insufficient, the officer processing the request will so notify the person making the request and indicate the additional information needed. Every reasonable effort shall be made to assist in the identification and location of the records sought.
