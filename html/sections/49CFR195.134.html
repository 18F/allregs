CFR /  Title 49  /  Part 195  /  Sec. 195.134 CPM leak detection.

This section applies to each hazardous liquid pipeline transporting liquid in single phase (without gas in the liquid). On such systems, each new computational pipeline monitoring (CPM) leak detection system and each replaced component of an existing CPM system must comply with section 4.2 of API RP 1130 (incorporated by reference, see Sec. 195.3) in its design and with any other design criteria addressed in API RP 1130 for components of the CPM leak detection system. [Amdt. 195-62, 63 FR 36376, July 6, 1998, as amended by Amdt. 195-99, 80 FR 186, Jan. 5, 2015]
