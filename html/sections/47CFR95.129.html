CFR /  Title 47  /  Part 95  /  Sec. 95.129 Station equipment.

Every station in a GMRS system must use transmitters the FCC has certificated for use in the GMRS. Transmitters that have been certified for use in the GMRS may be found on the FCC Web site at https://apps.fcc.gov/oetcf/eas/reports/GenericSearch.cfm. All station equipment in a GMRS system must comply with the technical rules in part 95. [80 FR 53752, Sept. 8, 2015]
