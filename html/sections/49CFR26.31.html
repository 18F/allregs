CFR /  Title 49  /  Part 26  /  Sec. 26.31 What information must you include in your DBE directory?

(a) In the directory required under Sec. 26.81(g) of this Part, you must list all firms eligible to participate as DBEs in your program. In the listing for each firm, you must include its address, phone number, and the types of work the firm has been certified to perform as a DBE.

(b) You must list each type of work for which a firm is eligible to be certified by using the most specific NAICS code available to describe each type of work. You must make any changes to your current directory entries necessary to meet the requirement of this paragraph (a) by August 26, 2011. [76 FR 5096, Jan. 28, 2011]
