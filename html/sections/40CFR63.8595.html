CFR /  Title 40  /  Part 63  /  Sec. 63.8595 How do I conduct performance tests and establish operating limits?

(a) You must conduct each performance test in Table 4 to this subpart that applies to you.

(b) Before conducting the performance test, you must install and calibrate all monitoring equipment.

(c) Each performance test must be conducted according to the requirements in Sec. 63.7 and under the specific conditions in Table 4 to this subpart.

(d) You must test while operating at the maximum production level.

(e)(1).

(f) You must conduct at least three separate test runs for each performance test required in this section, as specified in Sec. 63.7(e)(3). Each test run must last at least 1 hour.
