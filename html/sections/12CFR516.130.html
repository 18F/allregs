CFR /  Title 12  /  Part 516  /  Sec. 516.130 Where are comments filed?

A commenter must file with the appropriate OTS Regional Office (See table at Sec. 516.40(a)(2)). The commenter must simultaneously send a copy of the comment to the applicant. [66 FR 13003, Mar. 2, 2001]
