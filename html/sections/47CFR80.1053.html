CFR /  Title 47  /  Part 80  /  Sec. 80.1053 Prohibition on certification, manufacture, importation,

The manufacture, importation, or sale in the United States of Class A, Class B, Class S, or INMARSAT-E EPIRBs is prohibited. New Class A, Class B, Class S, or INMARSAT-E EPIRBs will no longer be certified by the Commission. [76 FR 67616, Nov. 2, 2011] Sec. Sec. 80.1055-80.1059 [Reserved]
