CFR /  Title 45  /  Part 1308  /  Sec. 1308.1 Purpose.

This rule sets forth the requirements for providing special services for 3- through 5-year-old children with disabilities enrolled in Head Start programs. These requirements are to be used in conjunction with the Head Start Program Performance Standards at 45 CFR part 1304. The purpose of this part is to ensure that children with disabilities enrolled in Head Start programs receive all the services to which they are entitled under the Head Start Program Performance Standards at 45 CFR part 1304, as amended.
