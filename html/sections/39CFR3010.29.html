CFR /  Title 39  /  Part 3010  /  Sec. 3010.29 Maximum size of Type 1-B rate adjustments.

Unused rate adjustment authority used to make a Type 1-B rate adjustment for any class in any 12-month period may not exceed 2 percentage points. [78 FR 52704, Aug. 26, 2013, redesignated at 79 FR 33834, June 12, 2014]
