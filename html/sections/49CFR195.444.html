CFR /  Title 49  /  Part 195  /  Sec. 195.444 CPM leak detection.

Each computational pipeline monitoring (CPM) leak detection system installed on a hazardous liquid pipeline transporting liquid in single phase (without gas in the liquid) must comply with API RP 1130 (incorporated by reference, see Sec. 195.3) in operating, maintaining, testing, record keeping, and dispatcher training of the system. [Amdt. 195-62, 63 FR 36376, July 6, 1998, as amended by Amdt. 195-99, 80 FR 188, Jan. 5, 2015]
