CFR /  Title 19  /  Part 149  /  Sec. 149.6 Entry and entry summary documentation and Importer

If the Importer Security Filing is filed pursuant to Sec. 149.2 of this part via the same electronic transmission as entry or entry/entry summary documentation pursuant to Sec. 142.3 of this chapter, the importer is only required to provide the following fields once to be used for Importer Security Filing, entry, or entry/entry summary purposes, as applicable:

(a) Importer of record number;

(b) Consignee number;

(c) Country of origin; and

(d) Commodity HTSUS number if this number is provided at the 10-digit level.
