CFR /  Title 7  /  Part 97  /  Sec. 97.11 Application accepted and filed when received.

(a) An application, if materially complete when initially submitted, shall be accepted and filed to await examination.

(b) If any part of an application is so incomplete, or so defective that it cannot be handled as a completed application for examination, as determined by the Commissioner, the applicant will be notified. The application will be held a maximum of 3 months for completion. Applications not completed at the end of the prescribed period will be considered abandoned. The application fee in such cases will not be refunded. [58 FR 42435, Aug. 9, 1993, as amended at 60 FR 17189, Apr. 4, 1995]
