CFR /  Title 34  /  Part 76  /  Sec. 76.142 An amendment is approved on the same basis as the

The Secretary uses the same procedures to approve an amendment to a State plan--or any other document a State submits--as the Secretary uses to approve the original document. (Authority: 20 U.S.C. 1221e-3 and 3474)
