CFR /  Title 14  /  Part 71  /  Sec. 71.51 Class C airspace.

The Class C airspace areas listed in subpart C of FAA Order 7400.9Y (incorporated by reference, see Sec. 71.1) consist of specified airspace within which all aircraft operators are subject to operating rules and equipment requirements specified in part 91 of this chapter. Each Class C airspace area designated for an airport in subpart C of FAA Order 7400.9Y (incorporated by reference, see Sec. 71.1) contains at least one primary airport around which the airspace is designated.
