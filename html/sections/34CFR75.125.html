CFR /  Title 34  /  Part 75  /  Sec. 75.125 Submit a separate application to each program.

An applicant shall submit a separate application to each program under which it wants a grant. (Authority: 20 U.S.C. 1221e-3 and 3474) [45 FR 22497, Apr. 3, 1980. Redesignated at 45 FR 77368, Nov. 21, 1980, as amended at 52 FR 27803, July 24, 1987; 60 FR 46493, Sept. 6, 1995]
