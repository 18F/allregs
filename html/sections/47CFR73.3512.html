CFR /  Title 47  /  Part 73  /  Sec. 73.3512 Where to file; number of copies.

All applications for authorizations required by Sec. 73.3511 shall be filed at the FCC in Washington, DC (Applications requiring fees as set forth at part 1, subpart G of this chapter must be filed in accordance with Sec. 0.401(b) of the rules.) The number of copies required for each application is set forth in the FCC Form which is to be used in filing such application. [52 FR 10231, Mar. 31, 1987]
