CFR /  Title 42  /  Part 441  /  Sec. 441.402 State plan requirements.

If a State that is eligible to provide community supported living arrangements services as an optional Medicaid service to developmentally disabled individuals provides such services, the State plan must specify that it complies with the minimum protection requirements in Sec. 441.404.
