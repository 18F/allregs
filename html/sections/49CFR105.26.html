CFR /  Title 49  /  Part 105  /  Sec. 105.26 Obtaining records on file with PHMSA.

To obtain records on file with PHMSA, other than those described in Sec. 105.25, you must file a request with PHMSA under the Freedom of Information Act (FOIA) (5 U.S.C. 552). The procedures for filing a FOIA request are contained in 49 CFR part 7.
