CFR /  Title 29  /  Part 1602  /  Sec. 1602.9 Commission's remedy for employer's failure to file report.

Any employer failing or refusing to file Report EEO-1 when required to do so may be compelled to file by order of a U.S. District Court, upon application of the Commission. [31 FR 2833, Feb. 17, 1966]
