CFR /  Title 42  /  Part 600  /  Sec. 600.315 Certified application counselors.

The State may have a program to certify application counselors to assist individuals to apply for enrollment in the BHP and other insurance affordability programs. If the State chooses this option, the State must follow the procedures and standards for such a program set forth in the regulations at either 45 CFR 155.225 or 42 CFR 435.908.
