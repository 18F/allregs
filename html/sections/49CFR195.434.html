CFR /  Title 49  /  Part 195  /  Sec. 195.434 Signs.

Each operator must maintain signs visible to the public around each pumping station and breakout tank area. Each sign must contain the name of the operator and a telephone number (including area code) where the operator can be reached at all times. [Amdt. 195-78, 68 FR 53528, Sept. 11, 2003]
