CFR /  Title 37  /  Part 360  /  Sec. 360.15 Separate claims required.

If a party intends to file claims for both cable compulsory license and satellite carrier compulsory license royalty fees during the same month of July, that party must file separate claims with the Copyright Royalty Board. Any single claim which purports to file for both cable and satellite carrier royalty fees will be dismissed.
